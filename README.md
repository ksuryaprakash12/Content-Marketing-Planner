# Content Marketing Plan for GreenGrow Organics

## Project Objective
To design and analyze a content marketing plan for an agriculture-based B2C brand using beginner-friendly Python.

## Brand Overview
GreenGrow Organics is an imaginary organic vegetable brand promoting healthy eating and sustainable farming.

## Tools Used
- Python
- Pandas
- VS Code

## Project Features
- Content calendar creation
- Content pillar identification
- Platform-wise scheduling
- Basic performance analysis

## How to Run the Project
1. Install Python
2. Install pandas using:
   pip install pandas
3. Run content_planner.py
4. Run analytics.py

## Learning Outcome
- Understanding content marketing strategy
- Using Python for marketing planning
- Basic data analysis skills
